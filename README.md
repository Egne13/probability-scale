# Probability Scale of Real-World Events (Estonia / RMK context)

This project visualizes real-world events as a probability scale to help build intuition about how likely different events actually are.

Humans are generally bad at understanding probabilities, especially very small or very large ones. This visualization helps bridge that gap.

---

## 🌲 Context

The dataset includes Estonian and nature-related events inspired by real-world statistics and environmental context (including RMK-related themes such as forests and wildlife).

---

## 📊 Example Events

- Rain in Tallinn tomorrow
- Seeing a moose in Estonian forest
- Forest fire in Estonia
- Car accident probability
- Lottery jackpot
